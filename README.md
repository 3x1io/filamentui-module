# Admin One UI Theme for VILT

Admin One UI Build as a theme for VILT Stack

## Install

```bash
composer require queents/ui-module
```
Add Module to `modules_statuses.json` if not exists

```json
{
    "UI": true
}
```
Publish Assets

```bash
npm i & npm run build
```

OR

```bash
yarn & yarn build
```
